# Aegis

Self-hosted personal intelligence platform built on [OpenClaw](https://openclaw.ai). Aggregate financial, calendar, health, and social data — surfaced through AI agents via WhatsApp.

4 Docker services. 10 integrations. Zero public ports. Everything encrypted at rest.
