## What is Aegis?

Aegis runs on a single VPS with 4 Docker services:

- **OpenClaw Gateway** — the brain. Manages AI agents, scheduling (cron), LLM calls, WhatsApp delivery, web UI, and agent memory.
- **Data API** — encrypted persistence layer. Stores, encrypts, and retrieves data from 10 integrations.
- **PostgreSQL** — the database with pgvector for embeddings.
- **cloudflared** — zero-trust tunnel. No public ports exposed.

Everything runs behind Cloudflare Tunnel with zero public attack surface. Security is the #1 architectural constraint.

## Key Features

- **10 Data Integrations** — Plaid banking, Schwab investments, Canvas LMS, Google Calendar, Outlook, Garmin health, LinkedIn, X, and more
- **4 AI Agents** — Interactive assistant, background sync, daily briefings, and content generation
- **8 Skills** — Finance, calendar, LMS, health, social, content, briefing, and security
- **3 Security Hooks** — PII redaction, budget guardrails, and hash-chained audit logging
- **8 Scheduled Jobs** — Automated sync, briefings, content publishing, and secret rotation reminders
